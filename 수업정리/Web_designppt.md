####좋은 Web site의 조건
1. 사이트 유용성
2. 양질의 컨텐츠
3. 사용자의 편의성
4. 흥미성
5. 개인화
6. 사용자의 참여 유도


####기획 단계
문제파악 -> 문제분석 -> 목표설정 -> 해결방안 -> 계획수립 -> 실행
거시적 : 새로운 영역을 개척, 방향성
미시적 : 사이트 구조 설계등 구체적 업무

4. 실행 가능한 범위

   -> 객관적 자료 취합
   -> 평가기준을 가지고 진행
