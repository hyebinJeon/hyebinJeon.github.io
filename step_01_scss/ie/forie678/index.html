forie678.js

작성자 : Mr. 하늘

IE9 이전의 구버젼 IE 브라우저를 최신 표준에 맞게 업그레이드해주는 All-in-one JS 파일입니다.
forie678.js는 jquery 1.6.2 이상 버전이 필요합니다.

기능 (적용 브라우저)

  • PNG 파일의 투명도를 인식하게 합니다. (IE6)
  • CSS3를 사용 가능하게 합니다. (IE6, IE7, IE8)
  • HTML5 Markup을 사용 가능하게 합니다. (IE6, IE7, IE8)
  • HTML5 WebForm을 사용 가능하게 합니다. (IE6, IE7, IE8)

PNG 파일 테스트

배경이 PNG 파일입니다.

CSS3 테스트

CSS3가 적용된 모습입니다.

input[type="checkbox"]

div:empty

section
article
